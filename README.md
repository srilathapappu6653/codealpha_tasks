# Data Science Internship

Overview

This repository contains the projects completed during my Data Science Internship. These projects demonstrate my skills in data preprocessing, exploratory data analysis (EDA), data visualization, machine learning, and predictive modeling using Python.

---

# Task 1: Iris Flower Classification

Objective

Classify Iris flowers into three species using machine learning.

Dataset

- Iris.csv

Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Output

Successfully classified Iris flowers with high accuracy using a Decision Tree Classifier.

Conclusion

The model accurately predicts the species of Iris flowers based on sepal and petal measurements.

# Task 2: Unemployment Analysis with Python

Analyze unemployment trends in India and study the impact of COVID-19.

Datasets

- Unemployment_Rate_upto_11_2020.csv

Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
